# 온기(Ongi) 프로젝트 문서

이 디렉토리는 Sprint별 작업 보고서 및 계획 문서를 보관합니다.

## 📁 디렉토리 구조

```
docs/
├── sprint-reports/          # Sprint별 완료 보고서 (Walkthrough)
│   ├── SPRINT_0_WALKTHROUGH.md
│   ├── SPRINT_1_WALKTHROUGH.md (예정)
│   └── ...
├── implementation-plans/    # Sprint별 구현 계획서 (예정)
├── design/                  # 디자인 관련 문서 (예정)
└── README.md               # 이 파일
```

## 📝 문서 종류

### Sprint Reports (완료 보고서)
각 Sprint 완료 후 작성되는 보고서로, 완료된 작업 내역과 검증 결과를 포함합니다.

### Implementation Plans (구현 계획서)
각 Sprint 시작 전 작성되는 상세 구현 계획서입니다.
